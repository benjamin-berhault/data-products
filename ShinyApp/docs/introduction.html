Introduction

This ShinyApp allows you to visualise Emergency Response Incidents data of New York City from this Data source. The data is made available under the NYC OpenData portal. I've been heavily inspired by the great job of Jo-fai Chow. If you are also interested by this kind of Shiny application, I highly recommend you checking his presentation http://bit.ly/londonr_crimemap.

The 2D Kernel density estimation is compute by the stat_density2d function using kde2d to display contours. (Chang, Winston. R Graphics Cookbook. pages 143-146, 2013)

  1. Select a date range
  2. Select an incident type
  3. Set custom properties that sounds good for you
  4. Click on the UPDATE GRAPH AND TABLES to update Heat Map, Trends and Sub Dataset tabs
  5. Enjoy the result!
